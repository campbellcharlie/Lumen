# File Jump Demo

This file demonstrates the file jump feature.

## How to Use

When you have many files open (more than 9), you can jump to any file by number:

### Quick Access (Files 1-9)
- Press `1` to jump to file 1
- Press `2` to jump to file 2
- ... and so on up to `9`

### Jump to Any File
- Press `:` to enter file jump mode
- Type the file number (e.g., `44` for file 44)
- Press `Enter` to jump to that file
- Press `Esc` to cancel

## Example Scenario

If you have 59 files open:
1. Press `:` - you'll see `:` in the status bar
2. Type `44` - you'll see `:44 (Enter file number)`
3. Press `Enter` - jumps to file 44

## Other Navigation

- `Tab` - Next file
- `Shift+Tab` - Previous file
- `:N` - Jump to file N
